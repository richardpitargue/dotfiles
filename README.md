## Requirements
- git
- stow
- zsh
- oh-my-posh
- alacritty
- tmux
- tpm
- neovim

## Installation

1. Clone this repo to your $HOME directory
```sh
git clone --depth=1 https://github.com/richardpitargue/dotfiles $HOME/dotfiles
```

2. Go inside the repository folder and use `stow` to create the symlinks
```sh
cd $HOME/dotfiles
stow .
```
